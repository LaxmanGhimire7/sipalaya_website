# Sipalaya Tech Academy - MERN Stack Platform (Still in Development Phase)

[![GitHub license](https://img.shields.io/github/license/LaxmanGhimire7/sipalaya_website)](https://github.com/LaxmanGhimire7/sipalaya_website/blob/main/LICENSE)
[![GitHub stars](https://img.shields.io/github/stars/LaxmanGhimire7/sipalaya_website)](https://github.com/LaxmanGhimire7/sipalaya_website/stargazers)

A full-stack educational platform built with MERN stack (MongoDB, Express, React, Node.js) offering professional Flutter development courses.

## Features
- 🚀 Full-stack course management system
- 📱 Responsive React frontend with modern UI
- 🔐 User authentication and authorization
- 🛒 Course enrollment system with payment integration
- 📊 Admin dashboard for content management
- 📚 Interactive learning environment with code editor
- 📧 Email notifications and support system

## Tech Stack
### Frontend
- **Framework**: React.js
- **State Management**: Context API
- **Styling**: CSS Modules, Flexbox/Grid
- **Routing**: React Router

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (with Mongoose ODM)
- **Authentication**: JWT, Bcrypt

### Tools
- **Package Manager**: npm/yarn
- **Version Control**: Git
- **API Testing**: Postman

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/LaxmanGhimire7/sipalaya_website.git
   cd sipalaya_website

## Install dependencies for both frontend and backend:
cd client && npm install
cd ../server && npm install

## Set up environment variables:
   Create .env files in both client and server directories with:

env
# Server .env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
PORT=5000

# Client .env
REACT_APP_API_BASE_URL=http://localhost:5000

## Start development servers:

bash
# In server directory
npm run dev

# In client directory
npm start
